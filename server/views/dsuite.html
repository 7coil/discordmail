Welcome to Dmail

A Discord approach to email.

Dmail is an experiment in a new kind of webmail, based on the idea that you should be able to communicate all in one place. The key features are:

  • Search, don't sort
    Discord's integrated search feature allows you to find the exact email you want, no matter when it was sent or recieved.
  • Don't throw anything away
    Storage is provided by Discord's excellent servers, so you'll never need to delete another message.
  • Keep the most important messages up top
    Pin messages you need the most using the right click options menu in Discord
  • No pop-up ads. No banners.
    Just the love of a free service.

Discord Mail Registration
Dmail does not currently support browser.
To register, type dmail register from within Discord, where a Dmail bot is available.
